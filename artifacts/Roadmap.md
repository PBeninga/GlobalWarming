# Release Goals

---

### Minimum Viable Product:
 - Full server client <br>
 - Server API <br>
 - Basic GUI <br>
 - Accounts <br>
 - Weak AI <br>
 - Single Player <br>
 
### Second Release:
 - Multiplayer<br>
 - Accounts<br>
 - Improve Graphics<br>
 - Basic map generation

### Third Release:
 - Migrate to production server
 - AI difficulty levels
 - Matchmaking
 - Finalize Graphics
 - Sound

### Final Release:
 - Start advertising, in app purchases
 - Make money
 - Retire to mansion
